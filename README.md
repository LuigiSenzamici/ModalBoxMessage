# Modal Box Input

Simply a Modal Box that display a message

## Getting Started
```bash
npm install modal-box-message --save
```
## Simple Use

get boxclass from module
```javascript
var boxClass = require("Modal-Box-Input").ModalBoxInput;
var Box = new boxClass("title", "message");
Box.Open();
```
